2. Grundsatz: Keine Datensammlung

Kurvenfokus wurde nach dem Prinzip "Privacy by Design" entwickelt:

  • Alle Daten bleiben auf deinem Gerät
  • Keine Registrierung oder Account erforderlich
  • Keine Analyse- oder Tracking-Tools
  • Keine Werbung
  • Keine Weitergabe an Dritte

3. Welche Daten verarbeitet die App?

3.1 Importierte GPS-Daten

Du kannst GPX- oder JSON-Dateien mit Motorrad-Touren importieren. Diese Daten:

  • Werden nur lokal auf deinem Gerät gespeichert
  • Werden niemals an Server übertragen
  • Können jederzeit von dir gelöscht werden

3.2 App-Einstellungen

Einstellungen wie Layout-Präferenzen werden lokal in den App-Einstellungen (UserDefaults) gespeichert.

3.3 Keine Live-Standorterfassung

Kurvenfokus erfasst keine Live-Standortdaten. Die App zeigt nur bereits aufgezeichnete Touren an, die du selbst importierst.

4. Berechtigungen

Die App benötigt folgende Berechtigungen:

  • Dateizugriff: Zum Importieren und Exportieren von Tour-Dateien

Die App benötigt keine Standort-, Kamera-, Mikrofon- oder Kontaktberechtigungen.

5. Karten-Darstellung

Für die Kartenanzeige nutzt die App Apple MapKit. Dabei gelten die Datenschutzrichtlinien von Apple.

6. Beta-Feedback (Website)

Während der Beta-Phase bieten wir auf unserer Website freiwillige, anonyme Feedback-Formulare an. Hierbei kommen folgende Technologien und Dienste zum Einsatz:

6.1 Eingesetzte Technologien

  • Google Firebase Firestore – Cloud-Datenbank zur Speicherung der Feedback-Einträge (Betreiber: Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland)
  • GitHub Pages – Hosting der Website (Betreiber: GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA)
  • Firebase JavaScript SDK – Client-seitige Bibliothek zur Übermittlung der Formulardaten

6.2 Welche Daten werden erhoben?

Die Feedback-Formulare erheben ausschließlich anonyme Daten:

  • Deine Antworten auf die Feedback-Fragen (z.B. Bewertungen, Auswahlen, Freitexte)
  • Zeitstempel der Absendung
  • Browser User-Agent (zur technischen Zuordnung)

Es werden keine personenbezogenen Daten erhoben – kein Name, keine E-Mail-Adresse, keine IP-Adresse, keine Cookies, kein Tracking.

6.3 Speicherort und Sicherheit

  • Die Daten werden in Google Cloud Firestore in der Region europe-west3 (Frankfurt, Deutschland) gespeichert
  • Die Datenbank ist durch Firebase Security Rules geschützt: Nur das Erstellen neuer Einträge ist möglich – kein Lesen, Ändern oder Löschen von außen
  • Der Zugriff auf die gespeicherten Daten erfolgt ausschließlich durch den Verantwortlichen über ein geschütztes Admin-Konto

6.4 Rechtsgrundlage

Die Verarbeitung erfolgt auf Grundlage deiner freiwilligen Einwilligung (Art. 6 Abs. 1 lit. a DSGVO) durch das aktive Absenden des Formulars.

6.5 Löschung der Feedback-Daten

Wichtig: Alle Beta-Feedback-Einträge werden nach dem offiziellen Produktstart von Kurvenfokus vollständig und unwiderruflich gelöscht. Die Daten dienen ausschließlich der Verbesserung der App während der Beta-Phase.

6.6 Auftragsverarbeitung

Für die Nutzung von Google Firebase besteht ein Auftragsverarbeitungsvertrag (Data Processing Agreement) mit Google gemäß Art. 28 DSGVO. Weitere Informationen: Firebase Datenschutz.
